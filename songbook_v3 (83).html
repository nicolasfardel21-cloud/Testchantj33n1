Format ChordPro — Guide de saisie

Accords

Placez l'accord entre [ ] juste avant la syllabe au-dessus de laquelle il doit apparaître. C=Do, D=Ré, E=Mi, F=Fa, G=Sol, A=La, B=Si.

[Bm]Je viens vers [G]toi [D]dans le se[Asus4]cret

Accords barrés : A/C# — Ligne d'accords seuls :

[D][A][Bm][G]

Métadonnées

Les directives marquées ✓ sont affichées dans l'en-tête, les autres sont silencieusement ignorées.

DirectiveAffichéePosition
{title: …}✓Titre principal (grand)
{artist: …}✓Sous le titre (petit)
{composer: …}✓Sous l'artiste (même taille)
{copyright: …}✓Sous le compositeur (très petit)
{key: …}✓À droite — tonalité de base
{tempo: …}✓À droite — BPM
{time: …}✓À droite — mesure
{subtitle: …}—Ignoré
{meta: …}—Ignoré

Un libellé personnalisé peut être ajouté après les deux-points pour tout type de section.

Exemple complet

{title: Je viens vers toi} {artist: Auteur} {key: Bm} {tempo: 75} {time: 4/4} {start_of_intro: Intro} [Bm][G][D][A] {end_of_intro} {start_of_verse: Couplet 1} [Bm]Je viens vers [G]toi [D]dans le se[Asus4]cret, [Bm]Loin du re[G]gard des [D]hommes, {end_of_verse} {start_of_chorus: Refrain} Ébloui [Bm]devant [G]toi, É[D]ternel, Je me pros[Bm]terne et j'a[G]dore {end_of_chorus}
